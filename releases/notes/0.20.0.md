## Highlights
- Added closed Island width tuning so external displays and different desktop layouts can keep the quiet Island footprint at a comfortable size.
- Very narrow closed Island widths now fall back to an icon-only presentation instead of squeezing mascot, counters, or session-rest text.
- Prepared Sparkle updates can now be installed immediately from Settings while keeping the existing automatic idle-time install path.
- The README lead now includes interaction demos so first-time visitors can see the docked notch and detached pet flows before installing.

## Fixes
- Fixed approval-style notifications that could remain after responding to high-risk QoderWork commands, matching issue `#214`.
- Fixed overlapping playback between notification sounds, bundled sounds, sound packs, and Settings previews by stopping the current sound before starting the next one.
- Fixed silent-width settings on notched displays being raised back to the physical notch width instead of honoring the user's selected value.
- Fixed Settings slider drags sometimes being interpreted as borderless-window drags.

## Notes
- The Sparkle build number is now `54`, and the app version is now `0.20.0`.
- This latest-`main` release focuses on closed Island sizing, update install control, notification cleanup, and consistent sound playback.
